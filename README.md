# Rapidly-Exploring Random Tree
## Simple RRT
Implemented a RRT in 2D domain, D=[0, 100]x[0, 100] with an initial configuration of (50, 50) and incremental distancee, d = 1. The reult is plotted for input number of vertices, K.
